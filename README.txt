Technical task:
Create a User class that describes a user (required fields: id, firstName, lastName, age, country). Create a list of a certain number of users.
Implement methods that accept a list of users and allow you to do the following:
1) display all users sorted by lastName;
2) display all users sorted by age;
3) check that for all users age > 7;
4) calculate the average age of all users;
5) display the number of different countries of residence (country) among the specified users

Task requirement:  implement all items using the Stream API.


*** THIS PROGRAM DOES NOT CONTAINS TOOLS OF INTERECTION BETWEEN USER AND SERVER, DOES NOT CONNECT WITH ANY SQL TABLE, DOES NOT IMPLEMENTS DAO SAMPLE

New tools that I used here : Lambda expressions, Stream API, Comparators, Lombok annotations
